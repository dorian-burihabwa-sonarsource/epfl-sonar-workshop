A method's return type should narrow down to match the potential value it matches

Using a more specific type helps the caller to check the return value.

Noncompliant Code Example

class MyClass {
  Object f(boolean cond) { // Noncompliant
    if (cond) {
      return "b";
    }
    return "a";
  }
}

Compliant Solution

class MyClass {
  String f(boolean cond) { // Compliant
    if (cond) {
      return "b";
    }
    return "a";
  }
}
